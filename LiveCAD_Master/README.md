# LiveCAD: Parametric Engineering Tool

## Quick Start
1. Run `setup.bat` (Installs Python dependencies)
2. Run `run.bat` (Launches the GUI)

## Git Integration
Run `git_setup.bat` to initialize this folder as a repository and push to GitHub.

## AI Assistant
Use the system prompt in `GEM_INSTRUCTIONS.txt` to train your LLM (Gemini/ChatGPT).
